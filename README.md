## 运行准备

安装JQuery
```
npm install jquery --save
```
在main.js文件中添加如下内容
```
import $ from 'jquery'
```
安装Bootstrap
```
npm install bootstrap --save
npm install --save popper.js
```
在main.js添加如下内容
```
import 'bootstrap/dist/css/bootstrap.min.css'
import 'bootstrap/dist/js/bootstrap.min'
```
